# yomi-KAI
## 概要
yomi-KAIはDiscordのテキストチャンネルに送られた文章をボイスチャンネルで読み上げるbotです。
## デモ
## 特徴
## 必要なソフトウェア
- Python 3.9 以上  
- ffmpeg 4.4 以上
## 必要なライブラリ
- discord.py[voice]  
- python-voicetext
## インストール
`pip install -r requirements.txt`を実行してライブラリをインストールしてください。  
次に、tokens.py.exampleのファイル名から.exampleを削除して、中にトークンを入力して上書き保存してください。
## 使い方
`python3 yomi-KAI.py`で起動できます。
